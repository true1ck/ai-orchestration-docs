02

Agent Role Definitions

Orchestrator

üß† Router / Bulk Orchestrator

The central dispatcher. Receives a raw ZIP payload, decompresses it to a secure sandbox, detects MIME types for dozens of files instantly, and spawns the exact right specialist agents in massive parallel streams. Collates and merges all outputs.

Modelclaude-sonnet-4-6
Input{ zip_buffer, timestamp }
Scaling50+ concurrent agents
PatternScatter-Gather / MapReduce
MCP Tool 1

üéô Transcriber + Cleaner

Triggered on every standard A/V file (mp4, mp3, wav). Runs Whisper (or calls API), then acts as a journalist editor‚Äîcleaning noisy transcripts, identifying speakers, and highlighting extreme sentiment shifts.

Inputfile_path: mp4/wav
Output{ transcript, key_quotes[], duration }
MCP Tool 2

üëÅ Vision Analyzer

Triggered on every image found inside the ZIP. Uses Claude's vision capabilities to describe the scene, extract visible text, and spot logos or landmarks without human review.

Inputimage_base64: str
Output{ description, text_ext, logos[] }
MCP Tool 3

üìÑ Document Extractor

Triggered on PDFs, DOCX, or unstructured web slices inside the dump. Uses OCR (if needed) and parses out people, organizations, dates, and a concise executive summary per document.

Inputfile_path: pdf/txt
Output{ summary, entities[], dates[] }
MCP Tool 4 & 5

üìä Analyst & üè∑ Categorizer

Runs sequentially as a post-processor on all text outputs (from videos, images, and documents). Scores bias, assigns taxonomy tags (politics, finance, etc.), and checks language.

Inputtext_output: str
Output{ sentiment, bias, tags[] }
MCP Tool 6

üîó Cross-File Aggregator

The crucial final step. Runs only when all parallel tasks finish. Scans the complete knowledge space to find intersection points‚Äîe.g. proving that the organization named in `doc_4.pdf` matches the logo found in `image_12.jpg`.

TriggerBarrier sync (all complete)
Output{ unified_report, cross_links[] }

Layer Tool / Library Purpose Notes

Document Extraction PyMuPDF / OCR Pares text layers from PDFs hidden in the payload Fallback to Tesseract if no text layer
AI Agents Claude Sonnet 4.7 Vision, extraction, and synthesis logic claude-sonnet-4-6 handles massive context
