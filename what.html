02

Agent Role Definitions

Orchestrator

üß† Router / Orchestrator

The central brain. Receives raw input with its type (video/article/image), determines which agents to call, in what order, and with what context. Merges all outputs into the final dashboard payload. Can loop Analyst back if confidence is low.

Modelclaude-sonnet-4-6
Input{ url, type, raw_text? }
OutputDashboardCard JSON
PatternSequential + conditional
MCP Tool 1

üéô Transcript Cleaner

Raw Whisper output is noisy ‚Äî filler words, speaker errors, run-on sentences. This agent cleans, segments, and optionally labels speakers. Also extracts key quotes suitable for the dashboard.

Inputraw_transcript: str
Output{ segments[], key_quotes[], duration }
ContextKeep small ‚Äî text only
MCP Tool 2

üì∞ Article Extractor

Parses article text and returns structured entities: headline, author, date, named people/orgs, key claims, and a 3-sentence summary. Designed to handle messy scraped HTML gracefully.

Inputarticle_text: str, url: str
Output{ summary, entities[], claims[], author }
ContextMedium ‚Äî long articles
MCP Tool 3

üëÅ Vision Analyzer

Uses Claude's vision capability to describe images: identify people (no facial recognition), text visible in image, logos, emotional tone, and whether the image supports or contradicts its article's headline.

Inputimage_base64: str
Output{ description, text_in_image, tone, logos[] }
Modelclaude-sonnet (vision)
MCP Tool 4

üìä Content Analyst

Core intelligence layer. Scores sentiment (‚Äì1 to +1), detects emotional tone, identifies potential bias signals, flags if content is opinion vs. fact, and generates a 1-line "media angle" description.

Inputtext: str, source_type: str
Output{ sentiment, tone, bias_flags[], angle }
RiskNeeds careful prompting
MCP Tool 5

üè∑ Categorizer

Assigns topics from a controlled taxonomy (politics, technology, finance, sports, entertainment, science, conflict, etc.) with confidence scores. Also detects language and region relevance.

Inputtext: str
Output{ tags[], language, region, confidence }
ContextTiny ‚Äî fast + cheap
MCP Tool 6

üîó Cross-Source Aggregator

Runs periodically across all processed items in session. Identifies which stories are being covered by multiple sources, detects narrative conflicts, surfaces emerging topics, and generates a session digest.

Inputall_items[]: DashboardCard
Output{ trending[], conflicts[], digest }
TriggerEvery 5 new items

Tech Stack

Layer Tool / Library Purpose Notes
Video Ingest yt-dlp Downloads audio/video from YouTube URLs pip install yt-dlp
Transcription OpenAI Whisper Speech ‚Üí text, multi-language, runs locally medium model recommended ~1GB RAM
Article Scraping newspaper3k Extracts article body, title, author from URLs handles paywalls poorly fallback: beautifulsoup
AI Agents Claude Sonnet 4.6 All agent calls ‚Äî vision, text, analysis claude-sonnet-4-6 single model for all agents
Orchestration MCP Python SDK Exposes agents as tools, handles calls pip install mcp stdio transport
Image Processing Pillow + requests Download, resize, base64 encode images Resize to max 1024px before sending to API
Backend API FastAPI REST endpoint: POST /analyze ‚Üí triggers pipeline async/await WebSocket for live updates
Frontend React + Tailwind Live dashboard: feed, sentiment charts, entity tags Polling or WebSocket for real-time feel
Storage SQLite ‚Üí Postgres Stores processed media cards + agent outputs SQLite fine for demo; Postgres for production
